🧠 HO‑DINA (Higher‑Order)

  • Estimasi mastery 8 atribut (A1–A8) dengan hierarki prasyarat (AHM)
  • Kovariat afektif θ‑afektif masuk ke struktur higher‑order → menggeser prior mastery
  • Implementasi: GDINA::GDINA(att.dist = "higher.order") via R/rpy2
  • Output: posterior mastery, guess/slip per butir, LRI

🤖 Neural CDM (NCDM) & XAI

  • Deep learning dengan Q‑matrix sebagai mask interaksi
  • Arsitektur: σ( disc · (θ - β) ⊙ Q ) + dense layers
  • SHAP/LIME → kontribusi atribut terhadap prediksi kesulitan
  • Terjemahan otomatis: “Kesalahan utama berasal dari atribut A2 (Pecahan), prasyarat aljabar”

📉 Learning Risk Index (LRI) & Rekomendasi Personal

Formula LRI:
LRI = 100 × [0.40·defisit + 0.35·breakage + 0.25·afektif]
defisit = rerata (1−mastery) · breakage = panjang rantai prasyarat putus · afektif = (θ‑afektif ternormalisasi)

Klasifikasi: 0–30 (rendah), 31–55 (sedang), 56–75 (tinggi), 76–100 (sangat tinggi).

Rekomendasi 3a‑3d: materi prioritas (hulu), materi ditunda, bentuk pembelajaran (CRA), level kesulitan adaptif + modulasi kecemasan.

🎯 Contoh output untuk Andi Pratama
  • Mastery terendah: A2 (0.35), A4 (0.30), A5 (0.25)
  • LRI = 59 → Risiko Tinggi
  • Rekomendasi: pulihkan pecahan (A2) dengan CRA, tunda persamaan linear
  • Dukungan afektif: kurangi tekanan evaluatif, kuis tanpa nilai
